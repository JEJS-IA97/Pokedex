# ⚡ Pokédex App

Una aplicación web moderna y ágil para explorar información de Pokémon. El proyecto está construido para ofrecer un alto rendimiento, estructurado con una arquitectura moderna que prioriza una excelente experiencia de usuario (UX/UI). 

🔗 **[Ver aplicación en vivo](https://pokedex-sable-iota.vercel.app)**

## ✨ Características Principales

* **Estética Minimalista:** Interfaz limpia sin superposiciones de texto innecesarias, enfocada en los datos y la accesibilidad visual.
* **Soporte de Visualización:** Estructura pensada para esquemas de alto contraste y adaptabilidad a modo oscuro.
* **Carga Optimizada:** Configuración basada en Vite para un empaquetado rápido y una experiencia fluida.

## 🛠️ Tecnologías Utilizadas

* **Frontend:** React, Vite
* **Estilos:** Tailwind CSS 
* **Despliegue:** Vercel
